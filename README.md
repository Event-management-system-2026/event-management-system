## Project Overview
EventFlow is a web-based Event Management System designed to help organizers create, manage, and track events through one centralized platform.

Many event organizers still rely on spreadsheets, messaging apps, or manual processes. These methods often lead to confusion, registration errors, and difficulty tracking participants. EventFlow aims to solve this problem by providing a simple and organized platform for both organizers and users.

## Main Features
### For users
- Browse available events
- View event details
- Register for an event
- Cancel participation
- Create accounts and log in

### For organizers
- Create new events
- Edit existing events
- Delete events
- Manage participant lists
- Track event information in one system

## MVP Goal
The MVP focuses on one complete working flow:

1. An organizer creates a new event  
2. The event appears in the system  
3. A user browses available events  
4. The user opens the event details page  
5. The user registers for the event  

This flow demonstrates the connection between the frontend, backend, and database.

## Main Technologies
- **Frontend:** React / HTML / CSS / JavaScript
- **Backend:** Node.js + Express
- **Database:** MongoDB
