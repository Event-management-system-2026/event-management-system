## Project Description
EventFlow is a web-based Event Management System designed to help organizers create, manage, and track events efficiently.

The system allows users to:
- browse available events
- register or cancel participation
- create accounts and log in

The system allows organizers to:
- create, edit, and delete events
- manage participant lists
- track event information in a centralized platform

## Main Technologies
- Frontend: React / HTML / CSS / JavaScript
- Backend: Node.js + Express
- Database: MongoDB

## Goal
To build a functional client-server event management system by the end of the semester that includes UI, backend, and database integration.
